# Zotero Library

Personal Zotero library management repository.

## Directory Structure

```
.
├── exports/      # Exported bibliography files (BibTeX, CSL JSON, RIS)
├── notes/        # Reading notes and annotations
├── styles/       # Custom CSL citation styles
├── attachments/  # Important attachment backups
└── scripts/      # Automation scripts
```

## Usage

- Export your Zotero library periodically to `exports/`
- Store reading notes in `notes/`
- Add custom citation styles to `styles/`
